    <h2 id="project-overview">Project Overview</h2>
    <p>The goal of this project is to demonstrate the development of a complete e-commerce website by converting a Figma design into a fully responsive Next.js application. It is suitable for deployment in hackathons or production environments.</p>

    <h2 id="features">Features</h2>
    <ul>
        <li><strong>Responsive Design:</strong> Mobile-first approach with responsiveness across all devices.</li>
        <li><strong>Dynamic Routing:</strong> Supports product detail pages and category-based product listings.</li>
        <li><strong>Reusable Components:</strong> Modular structure with reusable components like headers, footers, buttons, and sidebars.</li>
        <li><strong>State Management:</strong> Efficient state handling for cart, authentication, and user sessions.</li>
        <li><strong>API Integration:</strong> Simulated or real API integration for fetching product details and handling user actions.</li>
        <li><strong>E-Commerce Functionalities:</strong>
            <ul>
                <li>Product listing and filtering</li>
                <li>Product details with specifications</li>
                <li>Shopping cart</li>
                <li>Checkout process with payment integration</li>
                <li>Order confirmation</li>
            </ul>
        </li>
    </ul>

    <h2 id="pages-included">Pages Included</h2>
    <ul>
        <li>Home Page</li>
        <li>About Page</li>
        <li>Contact Page</li>
        <li>FAQ Page</li>
        <li>Product Listing Page (with sidebar filters)</li>
        <li>Product Details Page</li>
        <li>Blog Page</li>
        <li>Single Blog Post Page</li>
        <li>Login Page</li>

        <li>Order Confirmation Page</li>
    </ul>

    <ul>

        <li><strong>State Management:</strong> Context API / Redux</li>
        <li><strong>Icons & Fonts:</strong> Font Awesome, Google Fonts</li>
        <li><strong>API:</strong> Fetch API or Axios (mock or real API integration)</li>
        <li><strong>Build Tools:</strong> Webpack, Babel</li>
    </ul>

    <h2 id="getting-started">Getting Started</h2>
    <h3>Installation</h3>
    <ol>
        <li>Clone the repository:
            <pre><code>git clone https://github.com/your-username/your-repo-name.git</code></pre>
        </li>
        <li>Navigate to the project directory:
            <pre><code>cd your-repo-name</code></pre>
        </li>
        <li>Install dependencies:
            <pre><code>npm install</code></pre>
        </li>
    </ol>

    <h3>Running the Application</h3>
    <ol>
        <li>Start the development server:
            <pre><code>npm run dev</code></pre>
        </li>
        <li>Open your browser and visit <a href="http://localhost:3000" target="_blank">http://localhost:3000</a>.</li>
        <li>Build for production:
            <pre><code>npm run build</code></pre>
            <pre><code>npm start</code></pre>
        </li>
    </ol>
